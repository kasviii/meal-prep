# The Weekly Plate — Meal Prep Optimizer

A single-file web app that turns your diet, goals, and weekly schedule into a personalized 7-day meal plan, a grouped grocery list, and a batch-cooking timeline — generated by an AI model using your own free API key.

No installation, no server, no build step. Just open the HTML file in a browser.

## Getting started

1. **Open `meal-prep-optimizer.html`** in any modern browser (Chrome, Firefox, Edge, Safari) by double-clicking it, or upload it somewhere and visit it.
2. **Get a free API key** from one of:
   - [Google AI Studio](https://aistudio.google.com/apikey) — for Gemini
   - [Groq Console](https://console.groq.com/keys) — for Groq (Llama / GPT-OSS / Kimi models)

   Both have a free tier that's enough for generating plans regularly.
3. **Walk through the 5 onboarding steps** — diet & goals, allergies, tastes, your week, and connecting your API key.
4. **Click "Generate my plan."** It takes 10–30 seconds depending on the provider and model.

## What you get

- **Overview** — a short note about your plan, key stats, and a recap of your profile
- **Meal Plan** — a 7-day calendar; click any meal to see the full description
- **Grocery List** — consolidated and grouped by aisle, with checkboxes that save automatically; print it or download it as a `.txt` file
- **Prep Timeline** — what to batch-cook on which days so busy days take almost no effort

## Editing & regenerating

- **Edit profile** (Overview tab) takes you back through the wizard with your answers already filled in.
- **Regenerate plan** re-runs the same profile through the AI for a fresh plan without re-entering anything.
- **Start over** (top right, once you have a plan) wipes your saved profile, key, and plan from this browser.

## Your data

- Everything — your profile, API key, and generated plan — is stored only in your browser's `localStorage`, on your device. Nothing is sent anywhere except the AI provider you chose, and only when generating or regenerating a plan.
- Uncheck "Remember my key & answers on this device" before generating if you'd rather nothing be saved.
- Clearing your browser's site data, or using a different browser/device, will reset the app.

## Choosing a model

| Provider | Recommended model | Notes |
|---|---|---|
| Gemini | `gemini-flash-latest` | Always points at Google's current best Flash model |
| Groq | `openai/gpt-oss-120b` | Best quality on Groq's free tier |
| Groq | `openai/gpt-oss-20b` | Fastest, slightly lower quality |

You can also type a custom model ID for either provider if you want to try something else — check the provider's docs for current model names, since these change over time.

## Limitations to know about

- **Grocery quantities are AI-estimated.** The model sums ingredient amounts across the week itself; it's a solid starting point but not lab-precise. Glance over anything where an exact amount matters.
- **No persistent multi-device sync.** Since everything lives in one browser's local storage, your plan won't follow you to another device.
- **Output depends on the model.** If a plan looks off (ignored a preference, repeated a meal too often), regenerating often fixes it — or try switching models.

## Troubleshooting

- **"Couldn't generate your plan"** — usually an invalid/expired API key, a rate limit, or a model name that's been deprecated. Double-check the key and try the recommended model for your provider.
- **Blank or strange plan** — try regenerating; occasionally a model returns malformed output on the first attempt.
